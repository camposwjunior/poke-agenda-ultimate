# poke-agenda-ultimate
Pokéagenda Pro: App interativo com consulta à PokéAPI, filtragem das 9 gerações e cálculo dinâmico de fraquezas/vantagens. Inclui Dark Mode, sistema de tradução automática de descrições, áudio dos choros originais, modo Shiny e interface responsiva com animações CSS. Desenvolvido em JS Puro (ES6+) focado em performance e UX
